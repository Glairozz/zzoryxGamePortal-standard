# Game Hub

A web-based game portal that brings together curated browser games in one place. Browse, search, and discover games across multiple categories — all playable directly in your browser with no downloads required.

## Features

- **Game directory** with categorized listings (Strategy, Puzzle, Arcade, Educational, Party)
- **Search and filter** to quickly find games
- **Responsive design** that works on desktop and mobile
- **No downloads or sign-ups** — all games are web-based

## Tech Stack

- HTML, CSS, JavaScript
- Google Fonts (Inter)
- Fully static frontend

## Project Structure

```
├── index.html          # Landing page
├── games.html          # Game directory with search/filter
├── about.html          # About page
├── contact.html        # Contact page
├── css/
│   └── style.css       # Styles
├── js/
│   ├── games-data.js   # Game data
│   └── main.js         # Application logic
└── images/             # Image assets
```

## Usage

Open `index.html` in any modern browser. No build step or server required.
